#PoorSmith

##컨셉:
###초보 대장장이 컨셉. 재료 수급, 장비 제작, 판매까지.

##타겟 유저:
###경영, 크래프팅, 타이쿤 계열의 게임을 선호하는 유저.

##차별점:
###'초보' 대장장이가 컨셉인 만큼 필수적인 기능 소개를 위한 요소 이외에는 유저가 직접 발견해나가며 성장할 수 있는 시스템.

##스토리?:
###죽으면 대부분의 물건을 빼앗고 밖으로 내쫓는 마법의 숲에서 사냥을 하며 겨우 먹고 살던 사냥꾼. 빚을 내어 장비를 업그레이드 하고 일확천금을 노리는데... 몬스터들이 있는 동굴로 들어가다가 미끄러져 사망. 빈털털이가 됨. 가진 것은 엄청난 빚. 그래도 일은 다시 해야 되니까 근처에 떨어진 나뭇가지로 조잡한 나무 검을 만듬. 근데 지나가던 사람이 동전 하나를 앞에 툭 던지고 나무 검을 가져감. -> 아 이걸로 먹고 살아야겠다. -> 대장간 오픈!
